  • javascript

    1、查找DOM元素进行取值和赋值的操作、在特定节点位置进行内容的取值和赋值操作（innerHTML）。 2、针对元素进行事件监听。 3、通过操作DOM节点结合改变元素的CSS样式，达到绚丽的动画效果。 4、对DOM元素进行Ajax操作

    jQuery

    jQuery拥有强大的选择器，可以查找任意的DOM元素。同时jQuery对象实现链式语法，能够很方便简洁的写出强大的操作。比如操作属性;在页面加载时给DOM绑定事件。$("#chkbox").click(function(){ alert("hello jquery!"); });

    ajax

    基于web标准（standards-based presentation）XHTML+CSS的表示； 使用 DOM（Document Object Model）进行动态显示及交互； 使用 XML 和 XSLT 进行数据交换及相关操作； 使用 XMLHttpRequest 进行异步数据查询、检索； 使用 JavaScript 将所有的东西绑定在一起。
